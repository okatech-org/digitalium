# Prompt: iArchive + iAsted Implementation

> Ce fichier contient le prompt complet pour l'implémentation du système d'archivage intelligent.

## Résumé

**iArchive** : Système d'archivage numérique intelligent
**iAsted** : Assistant IA conversationnel (Intelligence Assistée pour Documents)

## Stack Cible (à adapter)

- Frontend : React + Vite + TypeScript
- Backend : Firebase (Firestore, Storage, Functions)
- IA : Anthropic Claude API (Sonnet 4)
- Localisation : XAF (FCFA), français, Gabon

## Catégories Légales

| Catégorie | Rétention | Base légale |
|-----------|-----------|-------------|
| Fiscal | 10 ans | Code Commerce Gabon |
| Social | 5 ans | Code du Travail |
| Juridique | 30 ans | Code Civil |

## Phases d'Implémentation

1. **MVP** (2-3 sem) : Upload, organisation, intégrité
2. **IA Core** (2-3 sem) : OCR, classification, iAsted chat
3. **Conformité** (2 sem) : Rétention, certificats, audit
4. **IA Avancée** (3 sem) : Extraction, insights, prédictions
5. **Collaboration** (1-2 sem) : Partage, annotations
